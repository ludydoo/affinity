Class: Header

Header

A Header is one of the two main parts of a Relation, along with the Body (which is a Set).

It is a Set of Attributes. It defines the structure of theTuples that will be part of the body of the Relation. Each Tuple in a Relation must match its Header

Example
var header = new affinity.Header([
     {firstName : {type : affinity.String}},
     {lastName : {type : affinity.String}}
])

var relation = new affinity.Relation(); // Create an empty relation without a header and without body

relation.header(header); // Sets the relation's header

relation.add({firstName : 'John', lastName : 'Doe'}); // Add a tuple in the Relation that matches the header
